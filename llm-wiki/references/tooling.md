# Optional tooling

Lazy-loaded. The skill works without any of these.

- **Obsidian** for browsing: graph view shows page connectivity;
  Dataview queries frontmatter for dynamic tables.
- **qmd** for search at scale (>100 pages): hybrid BM25/vector search
  with LLM re-ranking, CLI + MCP server.
- **Obsidian Web Clipper** for sources: browser extension converts
  articles to markdown for quick ingestion.
- **`skills/ingest-web-chat/`** for web LLM chats (Claude.ai, ChatGPT,
  Gemini, Le Chat) — CDP-driven, enterprise-SSO-safe.
